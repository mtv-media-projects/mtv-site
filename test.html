Зачем нужна эта страница?

Эта страница сделана для теста функций, которые не могут быть добавлены на все остальные страницы, по причине их нестабильности.

При условии, когда функция будет отлажена и проверена на все ошибки, то она будет добавлена на весь сайт.

На данный момент тестируются исправления мелких недочётов тёмной темы.
